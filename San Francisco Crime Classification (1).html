San Francisco Crime Classification-Neural Network Modeling

Data

This dataset contains incidents of crime described by the San Francisco Police Department from January 1st 2003 to May 13th 2015. The data was collected from the police department reports. The features of this dataset are Dates, Category, Descript, DayOfWeek, PdDistrict, Resolution, Address, X, Y.

Dates-timestamp of the crime incident.

Category-category of the crime. Category is the target variable that I am going to predict.

Descript-description of the crime incident.

DayOfWeek-the day of the week the crime occured.

PdDistrict-name of the police department district.

Resolution-how the crime incident was resolved.

Address-the approximate street address of the crime incident.

X-longitude

Y-latitude

I am going to check these features for collinearity and will remove the features with high colliniarity so that I don't overfit the data so I assume I will not be using all these features in my model building (unless there happens to be no collinearity).

I am going to be using different Neural Network models in order to predict the category of crime for this dataset. I am going to compare a Feedforward Neural Network with a Support Vector Machine (SVM) to see which type of deep learning neural network model or a machine learning model performs better based on classification accuracy and test loss.
